# 📊 Customer Churn Analysis

## 🚀 Project Overview

Customer retention is a key factor in the long-term success of any subscription-based business. This project analyzes customer subscription data to identify patterns associated with customer churn and provide insights that can help improve customer retention.

Using Python and Google Colab, the dataset was explored, analyzed, and visualized to understand how subscription duration and monthly spending relate to customer cancellations.

---

## 🎯 Objective

The objectives of this project are to:

* Analyze customer subscription data.
* Identify factors influencing customer churn.
* Explore the relationship between subscription length, monthly spending, and churn.
* Generate business insights using data visualization.
* Demonstrate a complete data analysis workflow using Python.

## 📋 Dataset Description

| Column              | Description                                               |

| Subscription_Length | Total duration of the customer's subscription             |
| Monthly_Spend       | Monthly amount spent by the customer                      |
| Churn               | Indicates whether the customer cancelled the subscription |
